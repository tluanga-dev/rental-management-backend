Load Predefined Packaging Types

About Predefined Data:

This will load a comprehensive set of standard packaging types from a predefined JSON file.

The predefined data includes common packaging types used across various industries:

  • Boxes - Various sizes and types of boxes
  • Pallets - Standard shipping pallets
  • Bags - Different bag types and materials
  • Containers - Various container types
  • Crates - Wooden and plastic crates
  • Drums - Industrial drums and barrels
  • Rolls - For rolled materials
  • Bundles - For bundled items

What will happen:

  • Existing packaging types will be updated with new information
  • New packaging types will be created
  • No data will be deleted
  • You'll see a summary of what was created/updated

Note:

This operation requires the predefined data file to be present at data/item_packaging.json.
